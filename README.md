# Elephy Landing (React + Vite)

Minimal conversion of your `Untitled-1.html` into a React landing using Vite.

## Requisitos
- Node.js 18+ recomendado

## Instalación (PowerShell)
```powershell
cd 'c:\Users\david\Escritorio\PROMO-Software\ELEPHY'
npm install
npm run dev
```

La app quedará disponible en `http://localhost:5173` (por defecto).

Si quieres que instale dependencias y arranque el servidor, dímelo y lo hago.
